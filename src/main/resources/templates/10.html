Unit 10: Recursion

Though Recursion may be a little confusing, essentially it is when a method calls itself repeatedly to solve a certain problem. There are some issues that can only be solved using recursion, and there are times when recursion is the most effective method of solving a problem.

Parts of a Recursive Method

Base Case:

The Base Case is the terminating case that is in a recursive problem. This case does not use recursion, and is simply a condition that causes the method to terminate. Without a base case, the recursive method would continue forever.

Recursive Case:

The Recursive Case is the set of steps that decrements as the Recursive Algorithm gets closer and closer to the Base Case

Recursive Algorithm:

The Recursive Algorithm is a finite set of steps that will call upon itself with simpler input values, this algorithm will eventually reach the Base Base.

Examples of Recursion

Factorials

One commonly used way of using recursion is to solve for factorials, here is an example:

    private int factorial(int n)
    
    {
    
      int res = 1;
    
      for(int i = n; i > 0; i--)
    
      {
    
          res *= i;
    
      }
    
      return res;
    
    }

Fibonnaci

Another Common use case for recursion is to create a fibonacci sequence generating algorithm, here is an example:

    private int fibonacci(int num)
    
    {
    
      // Base Case
    
      if(num == 0 || num == 1)
    
      {
    
          return 1;
    
      }
    
      // Recursive Function
    
      return fibonacci(num-1) + fibonacci(num-2);
    
    }

Searches

Binary Search

  • This is a much more efficient way of searching as it starts at the middle and decides to look for the greater half or the lesser half compared to the value that is being searched for
  • However the array must be organized in some way or the search will not work
  • Returns the index of the value that is searched for

    • Example:

        int binarySearch(int arr[], int l, int r, int x) { 
    
      if (r >= l) { 
    
          if (arr[mid] == x)
    
              return mid;
    
          if (arr[mid] > x)
    
              return binarySearch(arr, l, mid - 1, x);
    
          return binarySearch(arr, mid + 1, r, x);
    
      }
    
      return -1;
    
    }

    Sequential Search

  • The algorithm searches through each value in the array until the value is either found or it reaches the end of the array
  • An enhanced for loop can be used ONLY if you are spanning the entire array and not just a part of it
  • Returns the index of the value that is searched for

    • Example:
