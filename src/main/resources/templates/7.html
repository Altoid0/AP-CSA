Unit 7: ArrayLists

An ArrayList is a re-sizable array, also called a dynamic array. It grows its size to accommodate new elements and shrinks the size when the elements are removed. ArrayList internally uses an array to store the elements. Just like arrays, It allows you to retrieve the elements by their index.

Creating an ArrayList

In order to create an ArrayList, the steps are a little different than for normal Arrays, we must first declare and ArrayList with the following command ArrayList<type> varName = new ArrayList<type>();. In addition we must also specify the type, which cannot be primitive, so to have int type we must pass Integer and pass Double for double types. If we want to use strings, set the type to String. The following code snippet below shows an example arraylist of student grades being initialized and values added.

    // Create the ArrayList named `classGrades` and set its type to `Integer`
    ArrayList<Integer> classGrades = new ArrayList<Integer>();
    
    
    // Add the grades to our ArrayList
    
    classGrades.add(97);
    
    classGrades.add(50);
    
    classGrades.add(80);
    
    classGrades.add(90);

ArrayList Methods

Now we know how to create an ArrayList, but what can you actually do with Arraylists? Here we will be going over a few important methods that can be used on ArrayLists

Getting the Size of the ArrayList

With the .size() method, we can return the size of the ArrayList, or the number of elements in the ArrayList.

    int gradeSize = classGrades.size(); // Will return 4
    
    int classSize = students.size(); // Will return 3

Adding to an ArrayList

If we want to add to an ArrayList we can use the method .add(E obj), where the object E is added and returns True.

    classGrades.add(100);
    
    students.add("Trevor");

Inserting into an ArrayList

If we want to insert into an ArrayList, we can use the method .add(int index, E obj), where the object E is inserted at position index, and all the elements previously at position index and above are moved higher to the right and their indices are increased by 1, also adding 1 to the size.

    classGrades.set(1, 90); // Will change the value at index 1 to `90`
    
    students.set(0, "Ryan"); // Will change the value at index 0 to `Ryan`

Getting From an ArrayList

If we want to get a value from an ArrayList, we can use the .get(int index) method, which returns the element at index in the ArrayList.

    int grade = classGrades.get(2); // Will return `80`
    
    String student = students.get(1); // Will return `Wezley`

Setting in an ArrayList

If we want to set an element already in the ArrayList, we can use the method .set(int index, E obj), which replaces the element at the position of index with the specified object obj, and returns the element that was previously in that position index.

    classGrades.set(1, 90); // Will change the value at index 1 to `90`
    
    students.set(0, "Ryan"); // Will change the value at index 0 to `Ryan`

Removing From an ArrayList

If we want to remove an element from and ArrayList, we can use the .remove(int index) method, which removes the element from index and moves all elements higher than index to the left and subtracts 1 from the size of the Arraylist.

    classGrades.remove(1); // Removes `50` from `classGrades`
    
    students.remove(0); // Removes `Stephen` from `students`
