Comparing while loops to for loops

    for(int i = 0; i < 10; i++){}

Is the same as

    int i = 0
    while(i < 10){i++}

Foreach loops (Enhanced for loops)

The important thing to know about foreach loops is that they do not use indexes. Instead the variable i gets assigned to a variable inside the array and after that iteration is complete i will be set equal to the next element in the array. Since element values are being assigned to i we cannot manipulate i in order to modify an element within the array. For this reason only use foreach loops when you need to iterate the full length of an array and don't need to modify its content.

    for(int i : arrayofnumbers){
    // i is set to a number in the array each time}

It is worth noting that you don't have to iterate a collection of primitive type objects you can also iterate through an arraylist of any type you want.

Counting Iterations

If the center argument is i < some number we can do: some number - initial i value [10-0 = 10]

    for(int i = 0; i < 10; i++){}

If the middle argument has an “or equal to” operator then do the same as above but add 1 to the final number [10-0 = 10+1 = 11]

Nested Iteration

To find out how many times statement 1 will execute multiply the amount of times the outer loop will execute by the inner [10*5 = 50]

    for(int i = 0; i < 10; i++){
        for(int i = 0; i < 5; i++){
            //Statement 1
        }
    }
